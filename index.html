Git and Github tutorial

Following are the steps for the understanding of Version Control System

  1. Download git in your system Download from here
  2. Firstly make the empty git repository and initialize it by git init command on your terminal
  3. Check the status of your to add , delete or remove files to repository byu using git status Command
  4. After checking you can add a single file to staging area by using git add (filename) and multiple files by using git add .
  5. You can save the files residing in staging area by commiting the files by using git commit -m "My First Commit" you can write any message in double quotes
  6. Now you link your local repository to github repository git remote add origin URL
  7. To Push your code to linked repository you can use git push -u origin master / branch-name This is for first time you push code other wise just use git push
  8. If you change code on github you can easily fetch it your local repository by using git pull command it will fetch all the changed files to your local repository and now your both repositories are synchronized.
  9. One of the main features the branches is the main feature which allow you to make a sub repository without disturbing main stable epository by using git branch (branchname)
